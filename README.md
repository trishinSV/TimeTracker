# Общее описание

Требуется реализовать macOS приложение, которое блокирует доступ к выбранным пользователем приложениям на устройстве и подсчитывает количество попыток доступа к ним во время блокировки.

Приложение состоит из двух экранов.

## Экран "Blocked Apps"

На этом экране отображаются заблокированные приложения и кнопка для добавления новых приложений в список. Также предусмотрена возможность удаления приложений из списка.

При нажатии на приложение должен открываться экран статистики по приложению.

## Экран статистики по приложению

На этом экране отображается количество попыток открытия приложения. Также предусмотрена возможность временной паузы блокировки без удаления приложения из списка.

## Дизайн

Конкретные требования к дизайну отсутствуют, однако визуальная привлекательность идёт в учет при оценке. Продемонстрируйте свой вкус и умение работы с UI фреймворком.

# Технические требования

Помимо функциональных требований результат должен соответствовать ряду технических.

## SwiftUI как UI-фреймворк

Продемонстрируйте свои навыки структурирования и разработки SwiftUI-приложений. Пишите код с расчётом на будущее расширение проекта.

Может показаться избыточным для такой небольшой задачи, но, как и в случае со SwiftUI, закладывайте возможность дальнейшего расширения и продемонстрируйте свои навыки.
